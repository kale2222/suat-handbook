# 社团及书院兴趣小组

随着深理工学生人数不断增长，社团数量也在持续增加，希望大家都能在这里找到志同道合的伙伴。

如果没找到合适的社团，也可以自己向学校申请创办。

期待有一天能够看到百团大战的盛况。

:::info
学校非常支持社团发展，很多社团会不定期举办活动，部分活动还能获得**书院的经费支持**。
:::

## 书院兴趣小组

### 📚 曙光英语社

英语爱好者的聚集地。

### 🏔️ 曙光户外社

深圳的户外氛围很好。户外社之前举办了一个特色活动「**深圳十峰**」，这十座山是由深圳市民投票选出的。

### ⚽ 曙光足球社

足球社围绕**看球**和**踢球**展开，已经举办了非常多的活动：

- 去年组织观赛至少 5 次，包括欧冠决赛等
- 去年与深圳各校及单位开展了多场足球友谊赛
- 后续将举办更多元的趣味比赛
