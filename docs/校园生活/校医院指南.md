# 校医院指南

校内看病、买药、应急联系方式速查。

## 位置

天枢 3 号楼一楼后侧，**八分饱旁边**。

## 值班时间

| 日期         | 时间          |
| ------------ | ------------- |
| 周一至周五   | 9:00 – 18:00  |
| 周末 / 节假日 | 暂无固定值班  |

## 下班后怎么办

如果医生已经下班，但身体确实不舒服：

- 拨打**医务室门口张贴的电话**摇人
- 医生就住在楼上，能及时下来处理

:::info
紧急情况（突发剧痛、高烧不退、外伤等）请直接拨打 **120**，不要硬等到第二天。
:::
