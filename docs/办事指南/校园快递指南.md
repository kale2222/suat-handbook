# 校园快递指南

寄件填地址、取件找快递站，一篇全搞定。

## 寄到学校怎么填地址

「收件人」和「手机号码」填自己的，下面两栏照抄即可：

```
收件人：
手机号码：
所在地区：广东省深圳市光明区新湖街道
详细地址：公常路1号深圳理工大学
```

## 取件路线（三步走）

校园里的快递站同时也是**学校警务处**所在位置，按下面三步走即可。

### 第一步：从西门（正门）进入，沿主路

从深圳理工大学**西门即正门**进入校园，沿主路走到阶梯，**阶梯左侧**就是要走的方向（如图箭头处）。

<div style={{maxWidth: "80%"}}>

![阶梯左侧方向](./images/快递站1.webp)

</div>

### 第二步：从党群服务中心右侧入口进入

走到**深圳理工大学党群服务中心**，从其**右侧**进去（如图箭头处）。

<div style={{maxWidth: "80%"}}>

![党群服务中心右侧入口](./images/快递站2.webp)

</div>

### 第三步：到达快递站（即学校警务处）

进去就是快递站，凭取件码取件即可。

<div style={{maxWidth: "80%"}}>

![快递站内部](./images/快递站3.webp)

</div>

:::info
如果一不小心拿错了快递，告诉工作人员即可，他会处理。
:::
