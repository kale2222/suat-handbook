# 大学基础课学习

## 开学前要不要提前学？

有的同学可能在大一开学前想提前学点大学内容，我的建议是：**好好玩吧，完全没必要哈哈哈**

别的学校可能为了转专业需要卷，但深理工**自由选专业**，没有这方面的需求。

真要学的话，可以学一下怎么用 **AI 工具**——ChatGPT、Claude 这些，蛮有用也很普适。

---

## 数学

### 📐 高等数学

同济的教材略显抽象。如果看不懂，**不必妄自菲薄**，是书的问题，不是你的问题。

推荐资源：

- [**武忠祥高等数学零基础**](https://www.bilibili.com/video/BV1gA411j7Ta/?spm_id_from=333.337.search-card.all.click&vd_source=1cb514781558c00290b0823da055366b)（B 站，千万播放量）——我心目中最适合高中生的高等数学课。虽然是考研老师讲的，但真的非常适合萌新听。

### 📊 线性代数

推荐资源：

- [**宋浩线性代数**]B 站）——伟大无需多言
- [**3Blue1Brown · 线性代数的本质**]（B 站）——可视化理解
- [**MIT 线性代数**]（Gilbert Strang）——神中神！

:::tip
线性代数还蛮重要的——站在深度学习的角度，**比高数重要**。
:::

## 人工智能

### 🧠 深度学习

仅供参考：

1. **宏观入门**：[**3Blue1Brown 深度学习系列**](https://space.bilibili.com/88461692/lists/1528929?type=series)
2. **动手实战**：**「鱼书」**——《深度学习入门：基于 Python 的理论与实现》（斋藤康毅），高中生也能看懂的深度学习入门书
3. **学框架**：[**小土堆 PyTorch**](https://www.bilibili.com/video/BV1hE411t7RN/?spm_id_from=333.337.search-card.all.click)（B 站）
4. **体系课程**：**李宏毅深度学习** 或 **吴恩达深度学习** ——二选一即可
5. **补充阅读**：**李沐《动手学深度学习》**
