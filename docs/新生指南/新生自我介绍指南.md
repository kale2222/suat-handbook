# 新生自我介绍指南

有的同学可能对于自我介绍有一丢丢的恐惧，相比别人的”唱跳篮球”，自己没有太多”厉害”的爱好特长。

其实自我介绍这个东西，只要真诚大方就很好了，真诚大方是必杀技来着。

大概的框架就是：

- 姓名
- 家乡
- 爱好 / 特长
- 对于大学的期待
- 自己为什么选择深理工呀

都可以说的。

---

例如：

> “我叫 XXX，我来自湖南，大家对湖南的印象可能是红色文化以及非常出名的湖南卫视，例如快乐大本营呀。我的爱好是，XXXX 游戏，之所以喜欢玩这个游戏，是因为可以和朋友一起开黑。这个游戏有跨越空间，穿越时间的魅力。有的朋友很久没见面了，但是只要一上号，仿佛我们又回到了高中时代，很美好。至于为什么选择报考深理工，不知道，老师给我打电话我就来了。
>
> 很高兴认识大家，请多多关照～”
