# SUAT 手册

学生发起、面向深圳理工大学准新生 + 在校生的开源在线信息手册。

**在线访问**：https://suat-handbook.siat.workers.dev

## 本地开发

```bash
npm install
npm start       # 启动开发服务器 → http://localhost:3000
npm run build   # 构建生产版本
```

## 发布新版本

```bash
./scripts/release.sh v0.x "这个版本做了什么"
```
